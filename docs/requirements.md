# Pink Archive – Requirements (MVP)

## Business
- Product type: matching sets (sold as one set)
- Sizes at launch: S, M, L
- Shipping: UK only
- Payments: Stripe
- Launch: real domain, real customers, real orders

## MVP pages
- Home
- Shop (product grid)
- Product detail
- Cart (basic)
- Checkout (Stripe) (later)
- Order confirmation (later)
- FAQ (shipping/returns)
- Admin (add/edit products) (later)

## MVP success
- Admin can add a product (name, price, sizes/stock, image URL)
- Customer can browse products and open a product page
- Later: customer can pay via Stripe and order is saved
